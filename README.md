# 手機漫畫閱讀器
* 為了將一張一張斷掉的手機漫畫圖片，連接起來閱讀，而製作的工具。
* 漫畫圖檔名稱須為整齊含編號的名稱，例如：
    * `image_001.jpg`
    * `image_002.jpg`


### 功能介紹

#### 線上使用
* https://minruxie.github.io/comicReader

#### 使用方法
點擊「選擇檔案」按鈕，選擇「漫畫單話」資料夾，例如：第一話。

例如：
```
comicbook
  |- 第一話
  |   |- 001.jpg
  |   |- 002.jpg
  |- 第二話
```
